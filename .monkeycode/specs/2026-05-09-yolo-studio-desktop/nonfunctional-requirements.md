# 标炬（LabelTorch）非功能需求

## 稳定性

1. 长时训练期间 UI 不可冻结超过 3 秒。
2. 训练子进程异常不影响主进程存活。
3. 异常退出后可恢复最近项目上下文。

## 性能

1. 1000 张样本导入耗时在可接受阈值内。
2. 10000 张样本浏览分页无明显卡顿。
3. 标注编辑保存在批量场景可控。

## 可靠性

1. 标签写回采用原子替换。
2. 训练配置快照不可变。
3. 关键操作有审计记录。

## 可维护性

1. 分层清晰，禁止 UI 直连训练引擎。
2. 核心模块必须有单元测试。
3. 日志与错误码统一。

## 可移植性

1. Windows 为主目标平台。
2. 设计中保留 Ubuntu 兼容接口与路径抽象。
